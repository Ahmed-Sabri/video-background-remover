# Video Background Remover

This tool allows you to remove backgrounds from videos efficiently. It generates two output versions of the video:
1. **MP4 Format**: This version includes a green chroma background, which makes it ideal for further editing and compositing.
2. **WEBM Format**: This version features a transparent background, perfect for overlays and seamless integrations in presentations or live broadcasts.

## Features
- Quick and easy to use
- Supports various video formats
- High-quality output options

## Getting Started
To get started, simply upload your video, select your desired output format, and let the tool process your video background removal.
